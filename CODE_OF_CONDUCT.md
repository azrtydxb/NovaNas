# Code of Conduct

NovaNas is an open project. We ask contributors and participants to:

- Be respectful in discussions and reviews.
- Focus feedback on the work, not the person.
- Assume good intent; clarify before escalating.
- Follow the project's technical standards and review process (see [CONTRIBUTING.md](CONTRIBUTING.md)).
- Keep private information private; security reports go through [SECURITY.md](SECURITY.md).

Behavior that disrupts the project or makes participation unsafe may result in removal from project spaces. Enforcement is at the discretion of the maintainers.

Report concerns to `conduct@novanas.io` (placeholder — real address TBD). Reports are handled privately.

This code applies to all project spaces: repositories, issue trackers, pull requests, discussions, chat, and events.
